登录管理

获取验证码

接口地址

api/login/captcha

请求方式

GET

请求参数

无

响应参数说明
响应参数 参数说明
username 图片地址
uniqid 唯一标识 
{
    "code": 200,
    "message": "成功",
    "data": {
        "src": "/captcha/7272285ee62cfe5465e.html",
        "uniqid": "7272285ee62cfe5465e"
    }
}

访问验证码图片示例

http://localhost:8888/captcha/8960315ee626258f5d0.html

接口地址

请求方式

请求参数
请求参数 必传 参数说明
username ✅ 用户名
password ✅ 密码
code ✅ 验证码
uniqid ✅ 验证码唯一标识
