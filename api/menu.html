菜单管理

4. 菜单管理

新增菜单

接口地址

api/menu/addMenu

请求方式

POST

请求参数
请求参数 必传 参数说明
pid ✅ 父级菜单： 顶级菜单=0
name ✅ 名称
icon 图标
sort ✅ 排序
status ✅ 状态 0:下架 1：正常
url 菜单组件路径,如果status=1，url必传
remark 备注
响应示例
{
    "code": 200,
    "message": "成功",
    "data": ""
}

更新菜单，删除菜单

当status=0的时候视为删除菜单

接口地址

api/menu/updateMenu

请求方式

PUT

请求参数
请求参数 必传 参数说明
id ✅ 菜单ID
pid ✅ 父级菜单： 顶级菜单=0
name ✅ 名称
icon 图标
sort ✅ 排序
status ✅ 状态 0:下架 1：正常
url 菜单组件路径,如果status=1，url必传
remark 备注
响应示例
{
    "code": 200,
    "message": "成功",
    "data": ""
}

获取菜单列表

接口地址

api/menu/getMenuList

请求方式

GET

请求参数
请求参数 必传 参数说明
name 菜单名称
status 状态 0:下架 1:上架
page 页码
size 页数
